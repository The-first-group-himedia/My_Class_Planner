## 프로젝트 개요

- **목표**: 해외 대학생들을 위한 학습 관리 시스템(LMS) 개발


- **설명**: 해외 대학생을 위한 효율적인 학습 일정 관리를 위한 LMS 시스템


- **의의**: 학생들이 학업 일정을 효율적으로 관리하고 학습 계획을 세울 수 있도록 도움


- **배경**: 해외 대학생들의 학업 일정 관리를 개선하고 더 효율적인 시간 활용을 위해 시작


- **타겟층**: 현재는 미국을 타겟으로 개발 중

## 주요 기능

1. 회원가입/로그인
2. 시간표 조회
3. 시간표 생성기 (Time Table Generator)
4. 수강신청 쇼핑카트
